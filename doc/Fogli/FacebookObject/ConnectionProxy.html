Overview

The proxy object which sits between a facebook object and one of it's connections. This connection proxy only represents connections which are a "has many" type, which happens to be every connection represented by the Facebook Graph except for "picture" which is handled as a special case.

def parse_data(data)
  data ||= {}
  data["data"] ||= []
  data["data"] = data["data"].collect do |raw_item|
    connection_class.new(raw_item)
  end

  data
end
